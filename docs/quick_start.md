# Quick Start Guide

## Installation

```bash
pip install -r requirements.txt
```

## Run the App

```bash
streamlit run frontend/streamlit/app.py
```

Open browser to: `http://localhost:8501`

## Upload Data

1. Upload a `.fit` or `.gpx` file with heart rate data
2. View analysis results
